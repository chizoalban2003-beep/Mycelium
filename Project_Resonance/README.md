# Project Resonance Environment

This directory is the operational shell for a living digital ecosystem.

## Layers

- `bedrock/` — crystallized, stable logic and manifests.
- `liquid_flow/` — active, changing modules and orchestrators.
- `gaseous_noise/` — scratchpads, experiments, volatile ideas.
- `entropy_logs/` — chronological records of spikes, dissolves, and hardening.
- `secrets/` — dwellers-only heuristics and subconscious strategies.

## Workflow

1. **Perturbation**: inject an energy spike into liquid modules.
2. **Survival**: measure friction and dissolve weak paths.
3. **Sedimentation**: harden surviving motifs into bedrock manifests.

See `liquid_flow/settling_workflow.md` for the exact loop prompts.
